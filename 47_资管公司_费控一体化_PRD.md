# 费控一体化

## 原需求描述

1、自动识别发票真伪、抬头、金额、税率，自动查重，防范重复报销；自动归类费用，自动校验报销规则的合规性；
2、实时费用监控，超支自动预警，预算执行分析、差异自动归因。

---

## 页面结构

```
- 费控首页
  - 费用统计
  - 待审批

- 报销申请页
  - 发票上传
  - 报销填写

- 预算监控页
  - 预算执行
  - 超支预警

- 统计分析页
  - 费用分析
  - 差异归因
```

## 功能列表

### 1. 发票智能处理

#### OCR识别

```
[发票识别]

发票类型：增值税专用发票
识别结果：
├─ 发票代码：XXX
├─ 发票号码：XXX
├─ 开票日期：2024-01-15
├─ 金额：¥10,000
├─ 税率：6%
└─ 税额：¥566

[自动校验：通过]
```

#### 自动查重

- 重复报销检测
- 预警提示

#### 合规校验

- 报销规则校验
- 异常标注

### 2. 费用自动归类

- 费用类型识别
- 自动挂载科目
- 部门归集

### 3. 预算执行监控

- 实时数据更新
- 执行进度展示：
  ```
  [预算执行]

  部门：市场部
  年度预算：¥100万
  已执行：¥72万
  执行进度：72%
  预算余额：¥28万

  [超支预警] 执行进度超时间进度5%
  ```

### 4. 超支预警

- 阈值配置
- 预警推送
- 预警记录

### 5. 差异自动归因

- 差异计算
- 原因分析
- 改进建议

## 数据字段

| 字段 | 类型 | 说明 |
|------|------|------|
| invoice_id | string | 发票ID |
| invoice_type | enum | 发票类型 |
| amount | decimal | 金额 |
| tax_amount | decimal | 税额 |
| check_result | object | 校验结果 |
| expense_type | enum | 费用类型 |
| budget_id | string | 预算ID |
| execution_rate | float | 执行率 |
